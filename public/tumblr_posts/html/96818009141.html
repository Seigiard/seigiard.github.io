«Error: ENOENT open package.json» при npm install

Случается, при выполнении npm install в консоль вылазит что-то вроде нижеследующего:

 Error: ENOENT, open '/node_modules/.../package.json'

Ничего страшного, как оказывается, тут нет. Попробуйте почистить кеш и запустить заново:

npm cache clean # чистим кеш
npm install     # ставим заново
